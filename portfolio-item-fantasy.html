Times Series/WebScraping

Weekly fantasy point predictor

        I made this Dataset by creating a script in python that uses Selenium to scrape Espn.com for all the data for each player in the NFL. Data cleaning and preparation was performed to drop the players who have yet to play in a game. Feature engineering was used to split singles feature columns into multiple. Most players information was combined to train the same model expect for the defensive team beause it has stats that are so different then individual players. I also needed a way to show the model that certian player had 0 points because they were injured so I added a binary column to show if they were injured or not.

For my model I used xgBoost as used Mean Average Precision as the loss metric. After only 5 weeks of training data this season, I was able to acheive a MAP score of 5.8, which is great considering the volatility of football stats and games.
