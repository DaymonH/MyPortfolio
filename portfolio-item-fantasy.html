Times Series/WebScraping

Weekly fantasy point predictor

Data Collection/Data Ingestion

        I created this Dataset by creating various automated scripts in python, using Selenium, to scrape Espn and various other sites for all the data/stats for each player in the NFL each week. This data was loaded into the form of pandas dataframes and .csv files.

Data preprosessing

        Rigorous data cleaning and preparation was performed to drop the players who no longer play and have incomplete data. Feature engineering was used to split singles feature columns, that held multiple values, into separate columns.

Data storage

        Data was stored in a MySQL database on my local machine. Inicially I used PyMySql to connect to my DB and load all the data into is respective tables such as, players, teams, positions and game-stats. I also did the same using SSIS to create a package that would run the python scripts and load the data into the database. I then created a View that combined all the tables into one table for easy access and exporing.

Data Analysis and Exploration

        With this loaded dataset I performed Exploritory data analysis using various methods such as pythons plotly/seaborn.

Data Presentation

After exploring the data I found found various insights usefule to fantasy football enjoyers and created various dashboards using Tableau and Power BI where these insights could be seen and used.

Machine Learning

  • feature engineering
  • Model Creation/Finetuning
  • Continuous Model Training
  • Model Deployment

Most players information was combined to train the same model expect for the defensive team beause it has stats that are so different then individual players. I also needed a way to show the model that certian player had 0 points because they were injured so I added a binary column to show if they were injured or not.

For my model I used xgBoost and used Mean Average Precision as the loss metric. After only 5 weeks of training data this season, I was able to acheive a MAP score of 4.56, which is great considering the volatility of football stats and games.
